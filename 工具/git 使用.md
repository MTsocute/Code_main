# `git` 使用

---

<br>

## 1. 关于分支

| 操作             | 命令                                               |
| ---------------- | -------------------------------------------------- |
| 查看所有本地分支 | `git branch`                                       |
| 创建分支         | `git branch 分支名`                                |
| 切换分支         | `git checkout 分支名` 或 `git switch 分支名`       |
| 创建并切换分支   | `git checkout -b 分支名` 或 `git switch -c 分支名` |
| 推送分支到远程   | `git push origin 分支名`                           |
| 删除本地分支     | `git branch -d 分支名`                             |
